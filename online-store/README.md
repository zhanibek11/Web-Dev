# Online Store — Lab 4 Task 2

Angular приложение с каталогом товаров с Kaspi.kz.

## Запуск

```bash
npm install
ng serve
```

Открыть: http://localhost:4200

## Структура

- `src/app/models/product.model.ts` — интерфейс Product
- `src/app/components/product-list/` — список товаров
- `src/app/components/product-card/` — карточка товара

## Функциональность

- 10 реальных товаров с Kaspi.kz
- Галерея изображений с переключением через миниатюры
- Рейтинг в виде звёзд
- Кнопки шаринга в WhatsApp и Telegram
- Адаптивная сетка (CSS Grid)
